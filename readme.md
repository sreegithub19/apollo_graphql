Steps to run and deploy the project:

- npm install (or)
  - npm install --save apollo-server graphql @apollo/server express
  - npm install express-graphql --legacy-peer-deps
- node index.js
- git add . && git commit -m "C" && git push origin main && vercel --prod

---
